## K-Nearest Neighbors (KNN) – Iris Classification

## Description

 This project classifies iris flower species using the K-Nearest Neighbors (KNN) algorithm.

## Model

 K-Nearest Neighbors (KNN)
 Multi-class Classification

## Dataset

 Iris dataset (built-in from Scikit-learn)

## Workflow

* Load dataset
* Split into training and test sets
* Train KNN model
* Predict classes for test data

## Results

 The model classifies flower species based on feature similarity using distance metrics.

## Visualization

Scatter plot comparing:
  True labels
  Predicted labels

## Technologies

* Python
* Matplotlib
* Scikit-learn

##  How to Run

```bash
pip install -r requirements.txt
python main.py
```

##  Notes

This project demonstrates instance-based learning and classification using distance measures.
